📋 COMANDOS SSH PARA CONFIGURAR WEBHOOK CANAL 3000
========================================================

🔗 1. CONECTAR AO SSH:
ssh root@212.85.11.238

🔍 2. VERIFICAR STATUS ATUAL:
pm2 list
pm2 show whatsapp-3000
pm2 logs whatsapp-3000 --lines 20

🔧 3. CONFIGURAR WEBHOOK (TESTE TODAS AS OPÇÕES):

OPÇÃO A - PM2 ENV (MAIS PROVÁVEL):
pm2 set whatsapp-3000:WEBHOOK_URL "https://app.pixel12digital.com.br/painel/receber_mensagem_ana_local.php"
pm2 restart whatsapp-3000
pm2 logs whatsapp-3000 --lines 10

OPÇÃO B - LOCALIZAR E EDITAR ARQUIVO:
find /root -name '*.env' -o -name '*.json' | grep -i whatsapp
# Se encontrar arquivo .env ou config.json, editá-lo com:
# WEBHOOK_URL=https://app.pixel12digital.com.br/painel/receber_mensagem_ana_local.php

OPÇÃO C - RECRIAR PROCESSO:
pm2 delete whatsapp-3000
pm2 start app.js --name whatsapp-3000 --env WEBHOOK_URL="https://app.pixel12digital.com.br/painel/receber_mensagem_ana_local.php"

🧪 4. TESTAR:
curl http://localhost:3000/status
pm2 logs whatsapp-3000 --lines 5

📱 5. TESTE FINAL NO SEU COMPUTADOR:
php verificar_mensagens_recentes.php
# Envie mensagem WhatsApp para canal 3000
# Execute novamente: php verificar_mensagens_recentes.php

✅ "Webhook configurado para..."
✅ "Webhook URL set to..."
✅ "Listening on port 3000"
